Biomechanics Material Properties Analyzer (MATLAB)

Description: In biomechanical analysis, we utilize both destructive and non-destructive testing methods to assess values such as stress, strain, stiffness, and friction. These measurements are derived from key input values collected during testing: applied force, cross-sectional area, and change in distance. Additionally, we create visual representations, such as the stress-strain curve and stiffness curve, to enhance data analysis.

Purpose of this project: The overarching goal of this project is to optimize the efficiency of material property calculations and graph the obtained data based on a comprehensive set of input values. By enhancing the efficiency of biomechanical analysis, this project aims to have practical applications in real-life scenarios involving the human body. It can be employed to study and understand the behavior of elements like bones, muscle tissue, and other deformable objects under varying stress conditions.

Area of a polygon (MATLAB)

Description: In this project, the user must put in two inputs: the number of sides (numSides) and the length of each side (sidelength). The code first finds the perimeter of the polygon using the side lengths and number of sides. The code then finds the apothem using both inputs. Lastly, the code finds the area by multiplying the perimeter to the apthem and dividing the product by 2. If the area is impossible to calculate with the given inputs, the code will print the phrase "Invalid Format" and the user is allowed to give different input values.

Purpose of this project: The motivation behind this project stems from the extensive utilization of polygons in computer graphics, particularly in the creation of 3D shapes for animation and video games. To achieve a 3D structure, mutiple 2D polygons are interconnected at various angles. In summary, this project aims to create a robust and adaptable function capable of handling diverse polygon descriptions, validating their format, and providing accurate area calculations when appropriate.

Supervised Machine Learning Chat Bot

Description: My python code implements a simple chatbot using a knowledge base stored in a JSON file. The functions used in the code reads the knowledge base from the JSON file, saves modifications back to the JSON file, and finds the best matching answer to the question asked. The purpose of this code is to create a basic chatbot that learns and improves over time by allowing users to provide new answers for unrecognized questions. The chatbot leverages the knowledge base in the JSON file and employs a matching approach to find the closest question to the user's input. If a match is found, the corresponding answer is provided. If not, the bot requests the user to contribute a new answer, which is then added to the knowledge base for future interactions. This iterative learning process allows the chatbot to enhance its capabilities based on user input.

Purpose of this project: Machine learning plays a pivotal role in both online environments and real-life applications, exerting a profound influence that extends to optimizing various aspects of our daily lives. By harnessing and processing vast sets of data, machine learning algorithms have the capacity to transform the way we approach problems, make decisions, and enhance overall efficiency.

Cell Culture (Intro to Biomedical Engineering)

Description: In Biochemical Engineering (A branch in Biomedical Engineering), we worked with sterile cell culture to perform assays that measure pharmacodynamic outputs in living cells. In later weeks of lab, we worked with presto blue, which is metabolize by living cells to a pink product. Purpose of this project: To gain a hands-on experience in cell culture techniques, cell counting methodologies, and the assessment of cytotoxic drug effects on cellular viability. This knowledge is fundemental for reseachers and professors in pharmacology and cancer biology.

3D Printing (Academics and Hobby)

Description: 3D printers use a manufacturing process known as fused deposition modeling, which extrudes a thermoplastic filament through a hot nozzle.
